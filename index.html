Main.The content inside the "main" element should be unique to the document. It should not contain any content that is repeated across documents such as sidebars, navigation links, copyright information, site logos, and search forms. Note: There must not be more than one "main element in a document.Must NOT be a descendant of an "article", "aside", "header", "footer", "nav" element.
